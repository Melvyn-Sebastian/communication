# 🌉 CommBridge

> **Turn Awkward Moments Into Powerful Connections**

CommBridge is a free, real-life communication toolkit that helps you navigate tough conversations — whether it's a difficult talk with your boss, a sensitive discussion with a partner, or resolving workplace conflict. It guides you step by step with personalised scripts, strategies, and ready-to-use templates.

🔗 **Live Demo:** [communication-tau-jet.vercel.app](https://communication-tau-jet.vercel.app/)

---

## ✨ Features

- **50+ Real-Life Scenarios** — Browse curated situations across Work, Relationships, Family, and Social categories to find tailored guidance instantly.
- **Communication Wizard 🧙** — A step-by-step interactive tool (4 steps) that generates a personalised script and strategy based on your specific situation.
- **Golden Communication Rules** — Timeless principles that transform every conversation, presented as quick, digestible tips.
- **30+ Message Templates** — Ready-to-use scripts you can copy, customise, and send with confidence.
- **100% Free** — No sign-up, no paywalls, free forever.

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/your-username/commbridge.git

# Navigate into the project directory
cd commbridge

# Install dependencies
npm install
```

### Running Locally

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the app.

### Building for Production

```bash
npm run build
npm start
```

## 🌐 Deployment

This project is deployed on **Vercel**. To deploy your own instance:

1. Push your code to a GitHub repository.
2. Go to [vercel.com](https://vercel.com) and import your repo.
3. Vercel will auto-detect the framework and deploy.
